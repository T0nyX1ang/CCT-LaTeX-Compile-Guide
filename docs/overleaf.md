# 网络编译环境(Overleaf)

**该方法与Linux系统下本地XeLaTeX编译结果一致.**

!!! info "编译步骤"

- 访问[这个Overleaf页面](https://www.overleaf.com/read/skyzmpvnvvpr), 输入待编译的文档, 然后使用`XeLaTeX`编译

!!! tip "该方法的优点"

- 上手方便, 直接用就行

- 方便用于多人协作

!!! warning "该方法的不足"

- 由于Overleaf使用Linux系统进行编译, 字体可能与原生模板不一致

!!! appendix "额外的说明"

- 由于最终投稿需要使用`CCT-LaTeX`进行编译, 如果需要在文章中使用图片, 必须采用`eps`格式
